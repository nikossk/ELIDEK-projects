# ELIDEK-projects

## Overview
In this project we build a website simulating a databse of an institution called ELIDEK.

## Requirements
- MySQl for Windows
- Python, with the additional libraries :
  - Flask
  - Flask-MySQLdb
  - Flask-WTForms
